## 機能

クライアント接続断

- クライアントとサーバ間ではping-pongを行う
- ゲーム開始後の途中参加NG

ユーザマッチング機能
パケット中継

- 部屋は4部屋で
- 部屋はホスト1人と参加者3人で構成される
- 参加者は1人以上3人以下
